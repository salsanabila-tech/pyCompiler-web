Python Compiler Project

Compiler Python dari Source Code hingga Eksekusi

Proyek compiler Python sederhana yang dibangun untuk memahami tahapan utama compiler: Lexical Analysis, Parsing, Semantic Analysis, hingga Code Generation.

Memastikan output adalah y, variabel bebas x, dan operator valid.

Code Generation

Menampilkan bentuk fungsi hasil kompilasi tanpa menggunakan eval.

y

Execution

Menghitung nilai y dari range x dan menggambarkannya dengan Chart.js.

Tentang Proyek

                    

                        PyCompiler adalah proyek educational yang dibuat dengan Python untuk memahami
                        bagaimana sebuah compiler bekerja dari source code hingga execution.
